Adding a real-time feed to ArcMap

Purpose

This sample demonstrates connecting to a Tracking Server and adding one of the services to ArcMap. This sample also showsÂ initializing the Tracking Analyst extension.

Usage

  1. Open the solution.
  2. Change the TSSERVERNAME to the name of your Tracking Server machine.
  3. Change the TSSERVICENAME to the name of the real-time feed on your Tracking Server you want to connect to.
  4. Build the solution.
  5. Add the Add-in button to a toolbar in ArcMap.
  6. Click the Add-in button to have it add the temporal layer to Arcmap.
