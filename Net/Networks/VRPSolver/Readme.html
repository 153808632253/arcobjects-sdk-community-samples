Vehicle routing problem solver

Purpose

This sample application demonstrates how to programmatically perform a Vehicle Routing Problem (VRP) analysis. This sample shows how to loadÂ inputs, set solver parameters, execute the analysis, and obtain the output results.

Usage

  1. Use Visual Studio to build and run the application.
  2. In the application, click Find VRP Solution to perform the analysis. The VRP information is displayed.
  3. Optionally, change solver settings and re-solve.
