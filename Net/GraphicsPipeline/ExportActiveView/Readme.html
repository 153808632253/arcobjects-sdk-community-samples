Export active view

Purpose

ThisÂ sample shows how to export the active view to any supported export format. Functionality includes setting output quality, clipping to graphics extent, and disabling and re-enabling Microsoft font-smoothing or ClearType, using IPrintAndExport.

Usage

  1. Start Visual Studio, open the solution file, and compile the program.
  2. Start ArcMap and open an ArcMap document.
  3. Click the Customize menu, and click Customize Mode. The Customize dialog box opens.
  4. Click the Commands tab, and select the Developer Samples category.
  5. Drag and drop the Export active view tool icon onto a valid toolbar.
  6. Click the Export active view button to export the active view.
