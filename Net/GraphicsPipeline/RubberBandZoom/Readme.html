RubberBand zoom

Purpose

This sample demonstrates how to use the Display library's RubberBand object to zoom to the selected area.

Usage

  1. Start Visual Studio and open the applicable solution file.
  2. Build the project and start ArcMap.
  3. Click the Customize menu and click Customize Mode.
  4. Click the Commands tab and click Developer Samples.
  5. Click the applicable tool and drag it onto an active toolbar in ArcMap.
  6. Click the tool to enable it.
  7. Click and drag an area to zoom in. Right-click to go back to the previous extent.
