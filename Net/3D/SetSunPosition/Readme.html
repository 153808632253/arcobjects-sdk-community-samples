Set sun position tool

Purpose

This sample demonstrates manipulation of the globe light sourceâ€”enabling and disabling the light source, moving it as a response to the mouse position, and changing the sun's ambient light and contrast.

Usage

  1. Start Visual Studio, open the solution file, and build the project.
  2. Open ArcGlobe in design mode.
  3. Click the Customize menu, and click Customize Mode. The Customize dialog box opens.
  4. Choose the Add-Ins Controls category.
  5. Drag the Sun Position tool from the command items pane onto a toolbar.
  6. Run the application and choose the tool.
  7. Click the mouse to change the sun's position on the globe.
