Navigating around the SceneControl

Purpose

This sample demonstrates navigating around the SceneControl using the built-in scene navigation, rotate gestures, and 3D bookmarks.

Usage

  1. Browse and select a .sxd document to load into the SceneControl.
  2. To use enable rotate gestures also enable the navigate mode.
  3. If you have difficultly using rotate gestures; try reducing the ISceneViewer.GestureSensitivity property in code from the default 6 pixels to, for example, 2 pixels.
